# はじめに
ユニティちゃん2DデータのDemoをベースに作成した2D Run&Jumpゲームです。   
スペースキーでジャンプし、足場に触れるまでは2回までジャンプができます。  
ジャンプで足場に飛び移り、道中のUnityコインを取って高得点を狙おう！  

# 操作説明
スペースキー : ジャンプ  
Rキー : リトライ(GameOver時のみ)
